### Customer Churn Prediction

A machine learning model that predicts whether a bank customer will leave the bank.

Tech stack:
- Python
- Pandas
- Seaborn
- CatBoost
- Scikit-learn

Key steps:
- Exploratory Data Analysis
- Feature Engineering
- Model Training
- Feature Importance Analysis

Model performance:
- ROC-AUC: ~0.89
